# Codesys-ADS1115-i2c-Driver
This is a Codesys I2C driver for ADS1115 4 channel analog to digital converter
The library is a compiled library that is not editable. The editable version can be made available if requested

The Driver is composed of a device description file and a Library that implements the main function block that drives the i2c device

## Requirements
- Codesys IDE  v3.5 SP20 Patch 5 or higher
- Codesys Library
- I2C device description file

## Instructions
- Install the I2C device into your device repository, from the device description file
- Install the library into your library repository
- Check out a step by step instruction demo [from here](https://youtu.be/yYkmIfK0_Xg)
